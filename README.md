# About
**Mutopital** is a top down arcade style game where you have to select the right medicine for the right patient to heal their mutations. Mutopital was made for [ScoreSpace Jam# 28](https://itch.io/jam/scorejam28)

## Built with
- Unity 2021.3.16f1
